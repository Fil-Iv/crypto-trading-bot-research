## Crypto Trading Bot – Research Framework

Algorithmic cryptocurrency trading framework built for research, experimentation,
and demonstration purposes. The system focuses on adaptive strategies, risk control,
backtesting, simulation, and parameter optimization.

### Core Capabilities
- Modular and adaptive trading strategies
- Capital allocation and exposure control
- Backtesting and simulation engine
- Parameter optimization workflows
- ML / context-aware model selection
- Exchange abstraction with sandbox support
- Config-driven architecture
- Telemetry and notification modules

### Project Structure
- `main_bot.py` – main execution entry point
- `auto_bot.py` – automation and scheduling logic
- `utils/` – core trading framework modules (strategies, risk, models, exchanges)

### Security
API keys, secrets, and tokens are **not included** in this repository.
All sensitive configuration must be provided via environment variables.
